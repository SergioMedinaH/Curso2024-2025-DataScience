Evaluación de Requisitos de los Conjuntos de Datos

En este documento se evalúa si los conjuntos de datos seleccionados cumplen con los requisitos obligatorios (R1 a R4) y el grado de satisfacción de los requisitos opcionales (R5 y R6).

Requisitos Obligatorios

Requisito R1:

Los datos se encuentran en uno de los dominios de ciudad inteligente seleccionados

  • En nuestro caso el dataset seleccionado contiene los datos de las instalaciones municipales con zonas wifi gratuitas de la ciudad de MADRID.

Requisito R2:

Los datos se encuentran disponibles como un archivo CSV

  • El archivo del dataset es un CSV obtenido del portal de datos abiertos del Ayuntamiento de Madrid (216619-0-wifi-municipal.csv).

Requisito R3:

Los datos tienen una licencia abierta para que puedan ser publicados

  • Dentro de las Condiciones de uso del portal de datos abiertos del Ayuntamiento de Madrid aparecen lo siguiente:
    "Las condiciones generales permiten la reutilización de los documentos para fines comerciales y no comerciales. Se entiende por reutilización el uso de documentos que obran en poder del Ayuntamiento de Madrid, siempre que dicho uso no constituya una actividad administrativa pública.".

Requisito R4:

Los datos se pueden vincular fácilmente con entidades genéricas del mundo real (por ejemplo, ubicaciones)

  • El dataset con el que estamos trabajando, a parte del nombre de cada una de las instalaciones municipales con zonas wifi gratuitas, cuenta con información como la vía, el número ,código postal e incluso el barrio y el distrito en el que se encuentran, hasta es posible conocer las coordenadas geográficas de dichas instalaciones, esto facilita su localización y acceso mediante cualquier medio o plataforma digital.
    Otras entidades no tendrán problema en enlazarse con nuestra aplicación ya que nuestro dataset contiene mucha inforación de estos lugares.

Requisitos Opcionales

Requisito R5:

Existe documentación para los datos

  • Cuenta con documentación de la estructura del CSV que nos permite entenderlo con más facilicad.

Requisito R6:

Los datos pueden provenir de múltiples fuentes de datos.

  • En este requisito tenemos nuestras dudas pero pensamos que, aunque el dataset venga del portal de datos abiertos del Ayuntamiento de Madrid, es posible que hayan podido colaborar las distintas juntas municipales de los distintos distritos que constituyen la ciudad de Madrid.
